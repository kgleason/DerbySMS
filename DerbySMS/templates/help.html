Available commands

Anything inside { } will need to be replaced with an actual value. You don't need the { }


  • Place a bet: bet {shortname} {amount}
    • example: bet sn 13
  • Get your current status in the betting: status
  • Get the status of a horse: status {shortname}
    • example: status sn
  • Set up your name: name {firstname} {lastname}
    • example: name Jesse Duke

If you've been set up in the system as an admin, then the following commands are also available to you. Go ahead an try one. It will tell you if you are an admin or not.


  • Add a new horse: horse add {post position} {shortname} {longname}
    • example: horse add 1 lh Lulu Hogg
  • Modify an existing horse: horse mod {post position} {shortname} {longname}
    • example: horse mod 1 llh Lulu Hogg
  • Turn the bidding on: turn on
  • Turn the bidding off: turn off
